# עיצוב טקסט בפוסטים — תחביר

Markdown סטנדרטי + HTML גולמי מופעל (`markup.goldmark.renderer.unsafe: true` ב-`hugo.yaml`).

| אפקט | תחביר | תוצאה |
|------|-------|-------|
| מודגש | `**טקסט מודגש**` | **טקסט מודגש** |
| נטוי | `*טקסט נטוי*` | *טקסט נטוי* |
| קו תחתון | `<u>טקסט עם קו תחתון</u>` | טקסט עם קו תחתון |

## הערות
- **מודגש** ו-*נטוי* הם Markdown רגיל — עובדים תמיד.
- קו תחתון אין לו תחביר ב-Markdown, ולכן משתמשים בתגית HTML `<u>...</u>`. זה עובד כי `unsafe: true` מאפשר HTML גולמי בתוך התוכן.
- אפשר לשלב: `***מודגש ונטוי***`, או `**<u>מודגש עם קו</u>**`.
- מספרים/מונחים לטיניים בתוך עברית — עטוף ב-`<span dir="ltr">Sapere aude</span>` כדי שלא "יישברו" ב-RTL.
